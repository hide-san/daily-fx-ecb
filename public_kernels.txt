# public_kernels.txt
# ---------------------------------------------------------------------------
# Explicit allowlist of Kaggle kernel slugs that this pipeline is permitted
# to create and publish.
#
# Design intent
# -------------
# - Only slugs listed here are created and uploaded by the notebook pipeline.
# - Every listed slug is pushed with is_private=False (public).
# - A slug NOT in this file causes the create script to exit with an error,
#   preventing accidental creation of unknown or unreviewed notebooks.
# - To stop publishing a notebook: remove its slug from this list AND manually
#   set it to private via the Kaggle GUI.  Removing from this list only
#   prevents future re-creation; the existing Kaggle notebook is left as-is.
#
# When to edit this file
# ----------------------
# - Adding a new pair to pairs.txt: add the three per-pair slugs below.
# - Retiring a pair: remove its slugs (the Kaggle notebooks are left as-is).
# - Adding a new notebook kind: add its slug and update the create script.
# ---------------------------------------------------------------------------
